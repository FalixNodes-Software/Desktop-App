Don't have the miner software downloaded?



Instructions

  • Download the miner, if you haven't yet, and save it to a new folder to your desktop.
  • Before continuing, it is recommended that Powershell has be open as admin, please open Falix Software as admin.
  • In Powershell above, type cd Desktop\FolderName, replace FolderName with the name of the folder you made.

    • For CPU Mining:

  • In Powershell above, type falixnodes_miner-win.exe YourUserIDHere Threads, replace YourUserIDHere with your Discord ID and replace Threads with the number of threads you want to allocate for mining.

    • For GPU Mining:

  • In Powershell above, type falixnodes_gpu_miner-win.exe YourUserIDHere MaxGPUUsage, replace YourUserIDHere with your Discord ID and replace MaxGPUUsage with the max percentage of GPU usage you want from 1 to 100.
