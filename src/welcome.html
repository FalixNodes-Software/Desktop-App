Navigation

Navegación

导航

Use the sidebar to navigate between tabs. You can also use the controls in the Game Panel to navigate back and forth between pages and if you need to, you can also reload pages.

Utilice la barra lateral para navegar entre pestañas. También puede usar los controles en el Panel de juego para navegar hacia adelante y hacia atrás entre las páginas y, si lo necesita, también puede volver a cargar las páginas.

使用侧边栏在选项卡之间导航。 您还可以使用游戏面板中的控件在页面之间来回导航，如果需要，您还可以重新加载页面。

See a problem?

¿Ves algún problema?

看到问题了吗？

If you see any issues or bugs with our software, let us know right away by reporting it on GitHub. If you don't say anything, we may never become aware of the issue.

Si ve algún problema o error con nuestro software, infórmenos de inmediato en GitHub. Si no dice nada, es posible que nunca nos demos cuenta del problema.

如果您发现我们的软件有任何问题或错误，请立即通过在 GitHub 上报告来告诉我们。 如果你什么都不说，我们可能永远不会意识到这个问题。

Support

Apoyo

支撑

If you need help with anything else with our services, make sure to join our Discord server. It is also required to be on our Discord server to use our service.

Si necesita ayuda con cualquier otra cosa con nuestros servicios, asegúrese de unirse a nuestro servidor de Discord. También es necesario estar en nuestro servidor de Discord para utilizar nuestro servicio.

如果您需要有关我们服务的任何其他帮助，请确保加入我们的 Discord 服务器。 还需要在我们的 Discord 服务器上使用我们的服务。
