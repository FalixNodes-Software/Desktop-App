Autres paramètres d'apparence

其他外观设置

अन्य प्रकटन सेटिंग

Rounded corners can be enabled to mock round corners from Windows 11, this can be enabled on Windows 10 as well. Sidebar transpancy can be turned off, but the blur effect will stay in effect behind the app.

Les coins arrondis peuvent être activés pour simuler les coins arrondis de Windows 11, cela peut également être activé sur Windows 10. La transparence de la barre latérale peut être désactivée, mais l'effet de flou restera en vigueur derrière l'application..

Las esquinas redondeadas se pueden habilitar para simular esquinas redondeadas desde Windows 11, esto también se puede habilitar en Windows 10. La transparencia de la barra lateral se puede desactivar, pero el efecto de desenfoque permanecerá activo detrás de la aplicación.

可以启用圆角来模拟 Windows 11 中的圆角，这也可以在 Windows 10 上启用。 Sidebar transpancy 可以关闭，但模糊效果将在应用程序后面保持有效。

गोलाकार कोने विंडोज 11 से दौर कोनों उपहास करने के लिए सक्रिय किया जा सकता है, इस साथ ही Windows 10 पर सक्रिय किया जा सकता। साइडबार पारदर्शिता को बंद किया जा सकता है, लेकिन ऐप के पीछे धुंधला प्रभाव प्रभावी रहेगा।
